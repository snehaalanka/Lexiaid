# LexiAid — Listen, Understand, Empower

LexiAid is an accessibility-first web application that helps individuals with **dyslexia and reading difficulties** understand written content effortlessly.  
It converts **text, PDFs, and images** into **natural, human-like speech**, providing an inclusive way to access information anywhere.

> “Because everyone deserves to understand the world — one word at a time.”

---

## Features

- **Text-to-Speech:** Paste or type any text and listen instantly.  
- **PDF-to-Audio:** Upload a PDF and get it read aloud line by line.  
- **Image-to-Audio (OCR):** Upload or capture an image of text and hear it in seconds.  
- **Minimal, Accessible UI:** Clean and dyslexia-friendly interface with high contrast and intuitive flow.  
- **Web-based:** No installation is needed; it works directly in the browser.  

---

## Problem Statement

Reading barriers often limit access to education and information for people with **dyslexia, visual impairments, or learning disabilities**.  
Traditional screen readers can be overwhelming or hard to use. LexiAid bridges this gap by offering a **simple, lightweight, and emotionally supportive** way to convert written content into audio — instantly.

---

## Use Case

| User | Challenge | How LexiAid Helps |
|------|------------|------------------|
| A student with dyslexia | Struggles to read textbook PDFs | Uploads the file → listens to it in a friendly voice |
| A commuter | Wants to “read” notes while traveling | Copies text → converts to speech |
| An elderly person | Has trouble reading small text | Takes a photo → app reads it aloud clearly |
